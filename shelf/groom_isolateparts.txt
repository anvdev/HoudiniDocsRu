= Isolate Groom Parts =

"""Скрывает и отключает невыбранные объекты груминга для более быстрого отклика."""

Этот инструмент позволяет игнорировать невыбранные ноды [Guide Groom|Node:obj/guidegroom], объединенные с помощью [Groom Merge|Node:obj/groommerge]. Таким образом, генерация волос [Hair Generate|Node:/obj/hairgen] будет производиться только для выбранных нод. Данный инструмент заполняет параметр __Isolate Groom Objects__ в любой ноде [Groom Merge|Node:obj/groommerge], которая соотносится с выбранными нодами, в результате чего игнорируются невыбранные ноды. Если запустить этот инструмент со всеми соответствующими выбранными нодами [Guide Groom|Node:obj/guidegroom], то параметр __Isolate Groom Objects__ очищается, и генерация волос вновь производится для всех объектов.

Этот инструмент полезен для ускорения готовки сети и для удобного обозрения области, в которой вы сейчас работаете.

Для получения дополнительной информации см. справку [Работа с шерстью|/fur/workflow].

== Использование Isolate Groom Parts == (includeme)

    # Выберите [Guide Groom|Node:obj/guidegroom], над которым вы хотите работать.

    # Нажмите на инструмент [Icon:FUR/isolate_parts]__Isolate Groom Parts__ на вкладке __Hair Utils__. Это скроет объекты [Hair Generate|Node:/obj/hairgen], которые не используют выбранные направляющие.
    
    [Image:/images/fur/isolate.png]
    
    Этот пример показывает, что шерсть генерируется на голове, а не на туловище, поскольку перед использованием этого инструмента была выбрана только голова.
    
    TIP:
        Если вы выберете другой объект [Guide Groom|Node:obj/guidegroom] для работы и вновь запустите этот инструмент, то он скроет предыдущий и отобразит только новый.
