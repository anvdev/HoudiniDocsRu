= Create Guides =

"""Создает направляющие кривые на геометрии кожи или влияет на направляющие другого объекта груминга."""

Этот инструмент создает направляющие для персонажа. В отличие от инструмента [Add Fur|/shelf/groom_addfur], этот инструмент создает только лишь направляющие. Инструмент [Add Fur|/shelf/groom_addfur] создает направляющие, генерирует шерсть, а также дает возможность выбирать анимированную версию геометрии, в результате чего создается нода [Guide Deform|Node:obj/guidedeform].

Этот инструмент удобен при создании шерсти, так как большую часть времени ведется работа с направляющими, настраивается их начальное направление и длина. Также удобно создавать направляющие на отдельных частях персонажа, далее объединять их в один поток, а затем генерировать волосы для всего персонажа целиком.

Для получения дополнительной информации см. справку [Работа с шерстью|/fur/workflow].

== Использование Create Guides == (includeme)

    # Нажмите на инструмент [Icon:FUR/guidegroom]__Create Guides__ на вкладке __Hair Utils__.

    # Выберите геометрию кожи, на которой вы хотите добавить направляющие, и нажмите ((Enter)), чтобы подтвердить выбор.
    
        NOTE:
            Вы также можете выбрать существующий объект груминга.
    
    Если вы выберете геометрию кожи, [Guide Groom|Node:obj/guidegroom] сгенерирует направляющие кривые на геометрии.
        
    Если вы выберете какую-либо ноду с данными груминга, например другой [Guide Groom|Node:obj/guidegroom] или [Guide Deform|Node:obj/guidedeform], то с помощью созданного [Guide Groom|Node:obj/guidegroom] можно влиять на направляющие объекта или атрибуты его геометрии.
    
    Для получения справки по определенному параметру см. справку по ноде [Guide Groom|Node:obj/guidegroom].
