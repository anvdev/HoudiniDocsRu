= Plant Guides =

"""Позволяет вручную добавлять направляющие на поверхность персонажа."""

Этот инструмент позволяет вручную добавлять направляющие на поверхность персонажа. Можно добавить несколько направляющих, причесать их, и затем каждая новая направляющая будет интерполирована между существующими. При этом подходе можно работать с небольшим количеством волос, и при необходимости с легкостью изменить их или переместить в другие позиции. Инструмент можно использовать для небольшого количества волос, например, бакенбарды, или для персонажей, которые полностью покрыты волосами.

== Использование Plant Guides == (includeme)

    # Нажмите на инструмент [Icon:FUR/plant_guides][Plant Guides|Node:sop/guidegroom] на вкладке __Grooming__.

    # Добавьте направляющие, нажав ((LMB)) на геометрии персонажа.
    
    [Image:/images/fur/plantguides.png]

    TIP:
        Скроллинг колеса мыши или удерживание ((Shift)) при перетаскивании ((LMB)) позволит вам изменить размер кисти.
        
    NOTE:
        Plant Guides интерполирует длину и форму направляющих, основываясь на окружающих направляющих. Если отсутствуют какие-либо направляющие, то первая направляющая будет перпендикулярна поверхности кожи с длиной по умолчанию 0,1.

    Для получения подробной информации о параметрах см. справку [Guide Groom|Node:obj/guidegroom].
