= Guide Groom =

#type: node
#context: sop
#internal: guidegroom
#tags: groom, fur, stroke
#icon: FUR_shears

"""Интуитивное управление направляющими кривыми во вьюпорте."""

Эта нода позволяет интерактивно причесывать направляющие во вьюпорте. На вход может подаваться либо объект груминга, либо геометрия кожи.

@parameters

<!--
    Cache Strokes:
        Запекает все записанные мазки в кэш и удаляет мазки.
-->

Reset Strokes:
    Удаляет все записанные мазки и кэш, делая выход идентичным входной геометрии.

== Initial Setup ==

Primitive Group:
    Активная группа кривых.
    
Collision Radius:
    Настраивает расстояние выталкивания волос от геометрии кожи во время расчета столкновения.
    
Curve Segments:
    Количество сегментов в каждой направляющей. Когда эта нода создается с помощью инструментов на полке Grooming, то этот параметр связан с параметром Segment объекта Fur, так как это то, что будет использоваться во время рендеринга.

== Brush ==

Tool:
    Позволяет выбрать один из инструментов груминга.

    Каждый инструмент имеет соответствующий элемент на полке. Они подробно описаны в ссылках ниже.

    [Icon:FUR/screen_brush][Screen Brush|/shelf/screenbrush]:
            Причесывание волос в экранной плоскости.
            
    [Icon:FUR/surface_brush][Surface Brush|/shelf/surfacebrush]:
            Причесывание волос вдоль поверхности геометрии.
            
    [Icon:FUR/cut_hair][Cut Hair|/shelf/cuthair]:
            Интерактивное подрезание волос.
            
    [Icon:FUR/length][Extend Hair|/shelf/extendhair]:
            Интерактивное удлинение волос.
            
    [Icon:FUR/lift_hair][Lift Hair|/shelf/lifthair]:
            Интерактивное выталкивание волос от поверхности.
            
    [Icon:FUR/clump_hair][Clump Hair|/shelf/clumphair]:
            Интерактивная сборка волос в пряди.
            
    [Icon:FUR/part_hair][Part Hair|/shelf/parthair]:
            Интерактивное разделение волос от проведенной линии.
            
    [Icon:FUR/adjust_length][Adjust Length|/shelf/lengthadjust]:
            Интерактивное изменение длины кончиков волос.
            
    [Icon:FUR/smooth][Smooth Hair|/shelf/smoothhair]:
            Интерактивное сглаживание кривизны волос.

    [Icon:FUR/plant_guides][Plant Guides|/shelf/plantguides]:
            Интерактивное добавление отдельных направляющих.

Radius:
    Радиус кисти.

Strength:
    Настраивает силу эффекта различных кистей.

Lift:
    Обеспечивает минимальный угол подъема при расчесывании. Влияет только на те волосы, которые подняты выше, чем данное значение. Чтобы данный параметр работал правильно, радиус, указанный в параметре __Collision Radius__, должен охватывать все волосы.

Smoothing Radius:
    Радиус, в котором инструмент сглаживания ищет соседние направляющие.

Randomize:
    Значения, большие, чем ноль, будут случайным образом влиять на различные кисти.

Constrain Length:
    Выключение этого режима позволяет вытягивать волосы при расчесывании без каких-либо ограничений длины.

Lock Roots In Place:
    Выключение этого режима позволяет перемещать волосы и их корни по поверхности кожи.

Ray Bias:
    Смещает исходное положение лучей, используемых для проверки того, скрыты ли кривые за геометрией кожи. Это гарантирует, что лучи не будут случайно попадать в поверхности, из которых они происходят.

При использовании инструмента Adjust Length:

Target Length:
    Подгоняет при расчесывании длину волос к этому значению.

Random Length:
    Для каждой направляющей используется случайная длина между значением Min/Max.

Target Length Range:
    Диапазон значений, из которого случайным образом выбирается длина, при активном __Random Length__.

При использовании инструмента Plant Guides:

Interpolate Neighbors:
    Интерполировать длину и форму соседних направляющих при посадке новых.

Default Length:
    Когда __Interpolate Neighbors__ выключен, используется данная длина для вновь добавленных направляющих.

Ray Bias:
    Это смещение, используемое при проверке того, скрыты ли точки кривой за геометрией кожи. Учитываются только поверхности, которые перекрывают точку кривой на данное расстояние. Это делается для того, чтобы точки, близкие к поверхности, обращенной к зрителю, не считались скрытыми.

    Точки на обратной стороне геометрии толщина перекрытия которой меньше, чем это значение будут находиться под воздействием кисти.

=== Falloff ===

Ramp:
    Кривая убывания силы кисти от центра к краю. Кнопки сверху представляют собой небольшой набор пресетов.

=== Mirroring ===

NOTE:
    Отзеркаливание работает только тогда, когда выбранные направляющие с обеих сторон принадлежат ноде Curve Groom (Guide Groom).

Mirroring:
    Когда параметр включен, происходит отзеркаливание действий кисти при причесывании.
Origin:
    Начальная позиция зеркалирующей плоскости.
Direction:
    Поперечное направление.
