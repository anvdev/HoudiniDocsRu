= Draw Guides =

#type: node
#context: sop
#internal: drawhairguides
#icon: SOP/drawhairguides
#tags: fur, curve, stroke

[Include:/shelf/drawhair#includeme]

@parameters

Proxy Display:
    Показывает грубое представление формы волос для более быстрого обновления.

Skin Voxel Size:
    Определяет размер вокселя, используемый для VDB поверхности головы. Используется, чтобы избежать пересечений с кожей.

Type:
    Выбор между созданием полигона или кривой NURBS.

Segments Per Guide:
    Пересчитывает количество сегментов всех кривых.

Growth Field Quality:
    Определяет качество области, через которую перемещаются волосы. Минимальный требуемый размер вокселя вычисляется внутренне, исходя из радиуса нанесенных мазков. Удвоение значения качества вдвое уменьшает размер вокселя.

Export Groom Field:
    Экспортирует VDB вольюм вместе с направляющими кривыми.

== Drawing ==

Operation:
    Draw Guides & Groom:
        Создает волосы и изменяет область груминга.
    Groom Only:
        Изменяет область груминга не создавая волосы.

Radius:
    Настраивает радиус кисти.

Draw On:
    Skin:
        Рисование по геометрии кожи.
        
    Screen Plane From Root:
        Рисование волосами по экранной плоскости, начиная с определенной точки поверхности кожи.
        
    Surface Normal From Root:
        Рисование волосами перпендикулярно поверхности кожи, начиная с определенной ее точки.
        
    Screen Plane:
        Рисование волосами по экранной плоскости. В основном используется с опцией __Groom Only__.

Create Roots Along Stroke:
    Корни волос постоянно будут создаваться на поверхности кожи над которой совершается мазок, а не только в том месте, где произведено первое нажатие. 

Limit Curve Length:
    Ограничивает длину кривых в соответствии с данным значением. В комбинации с режимом __Groom Only__ позволяет быстро разместить короткие волосы на небольшой площади, что может быть использовано, например, при создании бровей.

== Root Creation ==

Skin Group:
    Группа примитивов на которой будут рисоваться волосы.

Display Roots Points Only:
    Показываются только корни волос, а не сами кривые, растущие из них.

Guide Spacing:
    Среднее расстояние между корнями направляющих.

Density Scale:
    Определяет плотность относительно расстояния между направляющими.

=== Density ===

Force Total Count:
    Игнорирует плотность направляющих и расстояние между ними, и просто разбрасывает данное количество.

Density Attribute:
    Если этот атрибут присутствует на геометрии кожи, то он используется для масштабирования плотности направляющих.

Emergency Limit:
    Никогда не генерируется больше, чем данное количество направляющих.

Relax Iterations:
    Выполняет данное количество повторяющихся релаксаций. Увеличение данного значения приводит к более равномерному распределению направляющих, но может замедлить производительность интерактивного рисования.

Percentage Of Guide to Use as Root:
    Разбрасывает корни по поверхности вдоль нарисованных кривых в указанном процентном отношении от их длины.

== Guide Advection ==

=== Shaping ===

Radius Scale:
    Множитель радиуса всех нарисованных кривых. Это глобальное значение и повлияет на все мазки.

Skin Normal Force:
    Множитель силы отталкивающей волосы от геометрии кожи.

Blend Width:
    Ширина области воздействия значения __Skin Normal Force__ от поверхности кожи.

Inner Blend Width:
    Ширина области воздействия значения __Skin Normal Force__ под поверхностью кожи.

Ramp Effect:
    Множитель эффекта кривой __Guide Shape__. Чем ближе это значение к нулю, тем меньше кривая воздействует на форму волос, таким образом делая их более прямыми.

Guide Shape:
    Задает форму кривой изменения радиуса от корней до кончиков волос для каждого мазка.

=== Advection ===

Advection Type:
    Метод интегрирования адвекции. Чем ниже выбранное значение, тем более точный результат, но и более дорогой.

Length:
    Множитель длины всех кривых.
    
    Увеличение длины будет работать только до того момента, до которого нарисованы мазки, поэтому значения выше одного обычно не работают.

Samples:
    Количество сабстепов, выполняемых во время адвекции.
    
    Настраивает точность адвекции, но не влияет на количество сегментов. Для этого используйте __Segments Per Guide__.

NOTE:
    Если вас интересует техническая характеристика техники адвекции волос, пожалуйста обратитесь к оригинальному выступлению на 2013 Siggraph [__Hair growth by means of sparse volumetric modeling and advection__ by Ashraf Ghoniem and Ken Museth, DreamWorks Animation|http://ken.museth.org/Publications_files/Ghoniem-Museth_SIG13.pdf].
