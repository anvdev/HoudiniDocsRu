== Curve Mask == (curvemask)

Use Curve Mask:
    #id: usecurvemask
    Управляет силой воздействия операции вдоль каждой кривой.

    Включает маску кривых. В выключенном состоянии все остальные параметры скрыты.

Range In Absolute Length:
    #id: curvemaskabsrange
    Во включенном состоянии параметры диапазона, находящиеся ниже, действуют в мировых единицах.

Range Min:
    #id: curvemaskrangemin
    Положение на кривой, с которого начинается эффект. Точки кривой, находящиеся перед этой позицией используют первое значение.

Range Max:
    #id: curvemaskrangemax
    Положение на кривой, на котором заканчивается эффект. Точки кривой, находящиеся за этой позицией используют последнее значение.

NOTE:
    Параметры. приведенные ниже позволяют быстро изменить общую форму кривой, и часто подходят для достижения необходимого эффекта, например, при воздействии только на корни или кончики волос. Изменение любого из них модифицирует форму параметра ramp и заменяет на ней любые ручные корректировки.

Effect Position:
    #id: curvemaskeffectpos
    Устанавливает позицию, в которой на кривую оказывается наибольшее воздействие. Корни находятся в позиции `0`, а кончики - `1`.

Falloff:
    #id: curvemaskfalloff
    Высокие значения приводят к широкой, колоколообразной форме параметра ramp. Низкие значения к заостренной форме. 

Influence Width:
    #id: curvemaskwidth
    Масштабирует значения манипуляторов в заданных позициях, чтобы обеспечить дополнительный контроль.

Curve Mask Ramp:
    #id: curvemaskramp
    Управление значениями маски кривой напрямую.



Source Mode:
    #id: sourcemode

    Groom Object:
	Загружает данные груминга из объекта груминга.

    Groom File:
	Загружает данные груминга из файла.

Groom Object:
    #id: sourcegroomobject

    Загружает данные груминга из указанного исходного объекта.

Groom File:
    #id: sourcegroomfile

    Загружает данные груминга из указанного файла.

== Skin VDB == (skinvdb)

Получение VDB образа геометрии кожи. Многие из инструментов в объекте Guide Groom, используют этот VDB вольюм для эффективного предотвращения проникновения сквозь поверхность кожи.

VDB Source:
    #id: vdbsource

    From Skin Geometry:
	Генерирует VDB из указанной геометрии кожи.

    SOP Geometry:
	Импортирует VDB из SOP контекста.

    File:
	Загружает VDB из файла.

Параметры режима __From Skin Geometry__:

Voxel Size:
    #id: voxelsize

    Генерирует VDB вольюм геометрии кожи, используя данный размер вокселя.

    Вольюм используется нодами SOP для отталкивания кривых от геометрии кожи.

Параметры режима __SOP Geometry__:

SOP Path:
    #id: vdbsop

    Импортирует VDB из указанного пути SOP.

Group:
    #id: vdbsopgroup

    Используемая группа VDB примитивов импортированной SOP геометрии.

Parameters for __File__ mode:

File:
    #id: vdbfile

    Загружает VDB образ поверхности кожи из файла по указанному пути.

Group:
    #id: vdbfilegroup

    Используемая группа файла.

TIP:
    Если SOP сеть или файл содержит несколько вольюмов, используйте `@name==surface`, чтобы получить только VDB поверхность, так как только этот тип вольюма используется инструментами груминга.
