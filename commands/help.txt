"""Печатает справочную информацию по использованию команды."""

:usage: `help [-k] [<<шаблон_команды>>]`

Без аргументов печатается список всех команд. Если шаблон задан, печатается список всех команд, которые соответствуют заданному шаблону. Если соответствует только одна команда или шаблон точно соответствует команде, будет напечатана подробная справка конкретно для этой команды.

@options
    `-k`:
        Ищет весь справочный текст для заданного шаблона.
