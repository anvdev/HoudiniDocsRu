"""Возвращает номер следующей точки подключенной к данной."""

Данная функция позволяет вам перебирать точки, подключенные к указанной точке (например, в том же полигоне). Обратите внимание, что "ближайшая" далее в тексте относится к подключению, а не к физическому расстоянию. Чтобы получить список всех подключенных точек сразу, используйте функцию [Vex:neighbours].

:usage: `int neighbour(<geometry>, int point_num, int neighbour_num)`

[Include:_common#geometry]

:arg:point_num:
    Номер точки, соседа которой вы хотите найти.

:arg:neighbour_num:
    Номер соседа для поиска. `0` - ближайшая подключенная точка, `1` - следующий ближайший сосед и т.д. Используйте функцию [Vex:neighbourcount] для получения общего количества подключенных точек.

:returns:
    Возвращает индекс точки соседней с заданной точкой. Порядок не определен, но будет согласованным для согласованной геометрии. Возвращает `-1`, если номер соседней точки вне диапазона, точка находится вне диапазона для данного входа или вход не существует.
